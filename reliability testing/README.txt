**********************************************************

Summary: We tested five treat dispensers 100 times each with 10 tests of increasing dispensing targets from 1 to 10 treats. Each row of the data file gives the expected and actual number of treats dispensed for a single trial.

Data files:
arce_stevens_2021_data.csv
 dispenser - dispenser number (1-5)
 expected - expected number of treats dispensed (1-10)
 actual - actual number of treats dispensed (1-10)

R code:
 arce_stevens_2021_rcode.R - code for running computations

Instructions to reproduce results:
 To reproduce these results, open arce_stevens_2021_rcode.R and ensure that all packages mentioned at the top of the script are installed.  Once all packages are installed, run the script in R using "source("arce_stevens_2021_rcode.R")".
